### Frontend

#### setting

```bash
npm create vite@latest <프로젝트 이름> -- --template react-ts

npm install react-globe.gl
npm install three @react-three/fiber @react-three/drei @types/three
npm install axios
npm install react-markdown
```

#### start
```bash
npm run dev
```
